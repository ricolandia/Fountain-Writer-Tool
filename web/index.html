    📖 Manual do usuário

    Cabeçalho de cena: INT. CASA - DIA ou .CENA

    Personagem: JOÃO (maiúsculas, sozinho na linha)

    Diálogo: linha abaixo do personagem

    Rubrica: (sorrindo) entre parênteses

    Transição: CUT TO: ou > FADE OUT

    Centralizado: > texto <

    Atos: # Ato 2 (marca mudança de ato)

    Sinopse: = resumo da cena (não aparece na impressão)

    Nota de produção: [[ nota ]] (não aparece na impressão)

    Comentário descartável: /* texto */ (ignorado)

    Forçar ação: ! texto

    Forçar personagem: @ nome

    ☁️ Sincronizar entre dispositivos

    O Fonte salva seus roteiros no formato .fountain.json. Para acessar de qualquer computador:

    1. Instale o Dropbox, OneDrive ou Google Drive no seu PC
    2. Crie uma pasta sincronizada (ex: Dropbox/Roteiros/)
    3. No Fonte, clique em 💾 Salvar e escolha essa pasta
    4. Na próxima vez, o 💾 salva automaticamente no mesmo lugar
    5. No outro computador, abra 📂 Abrir e selecione o arquivo

    O sincronismo é feito pelo próprio serviço de nuvem — o Fonte não precisa de servidor nem de login.

    📦 Zip universal
    Baixe o Fonte-web.zip na página de Releases do GitHub, extraia e abra index.html.

    📲 Instalar como app (PWA)

    O app funciona em qualquer navegador. Para instalá-lo como aplicativo:

    ✅ Chrome / Edge / Brave: ⋮ → Instalar
    ✅ Samsung Internet: ⋮ → Adicionar página a → Tela inicial
    ⚠️ Firefox Android: ⋮ → Adicionar à tela inicial
    ⚠️ Firefox Desktop: só funciona no navegador (sem botão Instalar)
    ⚠️ iOS (Safari): Compartilhar → Adicionar à tela de início
    ❌ Xiaomi (Mi Browser): não suporta PWA — use o Chrome pela Play Store
